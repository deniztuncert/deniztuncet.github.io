Projects

AutoElec — Automated EEG Electrode Localization (3D Scan → Electrode Coordinates)

Status: Released / v1 • Methodology tool • 3D mesh → electrode centers → labels

A pipeline that aligns a scanned head mesh using fiducials (e.g., Nz/LPA/RPA/Cz/Oz), extracts electrode candidates using surface curvature, clusters electrodes (DBSCAN), and labels electrodes by warping a 10–20 template to the scan.

Repo: github.com/deniztuncert/automated-eeg-localization
